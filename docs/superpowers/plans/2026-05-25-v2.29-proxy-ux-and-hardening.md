## Batch 3 — `engine.stop()` async 化（HX-3）

涉及：`server/engine.js`、`protocol-engine.js`、`server/routes/execute.js`

1. `PipelineEngine.stop()`：改 `async`；按顺序 await：
   - 把 abort flag 设上（保持现有逻辑）。
   - `await this._browser?.close().catch(()=>{})`
   - `this._chromeProc?.kill('SIGKILL')`，然后等 `chromeProc.exited` 或 setTimeout 兜底。
   - `await fs.promises.rm(this._tempDir, { recursive: true, force: true })`。
   - 最后 `this.status = 'idle'`。

2. `ProtocolEngine.stop()`：相同流程，但只杀 Python 子进程 + Chrome（如果支付阶段起了的话） + tempDir。

3. `server/routes/execute.js` 启动新 engine 之前：`if (engine) { try { await engine.stop(); } catch {} }`。同时关掉旧 `engine.logCapture?.stop()` 避免 HX-10 嵌套泄漏（即便不重写 LogCapture，至少显式 stop）。

4. 验证：跑 `node --test __tests__/protocol-engine.test.js` 等已有引擎测试，新增一个 stop 测试比较麻烦（要 mock Browser/Chrome），暂只做手动 smoke：UI 启动 → 等账号开始 → 点 Stop → 观察 Chrome 进程被杀干净（任务管理器 chrome.exe 应消失）。

## Batch 4 — LogCapture 兜底脱敏（HX-2）

涉及：`server/logger.js`、新增 `server/__tests__/logger-redact.test.js`

1. `server/logger.js`：
   - 新增模块级 `function redact(msg)`：
     - JWT：`/eyJ[\w-]+\.[\w-]+\.[\w-]+/g` → `eyJ…`。
     - access_token / refresh_token URL 参数：`/(access_token|refresh_token)=[^&\s]+/gi` → `$1=***`。
     - OTP / SMS 验证码：`/\b(otp|code|verification[ _-]?code)\b[^a-z0-9]{0,8}(\d{4,8})\b/gi` → `$1=***`（保留关键词，只脱敏数字）。
   - `LogCapture` 在 emit 给 listener / push 到 socket 之前，先把 message 过一遍 `redact`。`_originalLog(redacted)` 用脱敏后的版本（生成 server.log 也脱敏）。

2. 新增 `server/__tests__/logger-redact.test.js`：
   - 测试 JWT 三段被替换。
   - 测试 `access_token=abc.def` 被替换为 `access_token=***`。
   - 测试 "verification code 123456" 中 123456 被替换为 ***。
   - 测试不含敏感模式的字符串原样通过。

3. 验证：`node --test server/__tests__/logger-redact.test.js`。

## 提交粒度

每个 Batch 一个 commit，commit message 引用 spec 编号（如 `fix(proxy): PX-1 getState 不再回吐订阅 token`）。Batch 8~15 是前端，按 spec 编号 `feat(web): FX-1 ...` 风格。最后一个 commit 是 "test: 加 npm scripts.test + 3 个新单测"。

## 失败时回滚

- 任何 Batch 完成后跑 `npm test` 出红，立即 `git revert HEAD` 后逐项排查。
- 前端 build 失败立即 revert，不带病改下一项。
- 手动 smoke 找到回归（如 Stop 按钮卡死），立即 revert 涉及 Batch 后排查根因，再重做。
